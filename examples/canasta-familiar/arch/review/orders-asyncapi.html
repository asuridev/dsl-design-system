Orders Async API v1.0.0
Asynchronous messaging contracts for the `orders` Bounded Context of Canasta Familiar. **Published channels:** Events emitted by the orders BC to the message broker. **Consumed channels:** Events consumed by the orders BC from other BCs. All events are persisted via the Outbox pattern (outboxRetentionDays: 7). Consumer idempotency is enforced via a processed-events store (TTL: 14 days).
