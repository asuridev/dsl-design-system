sequenceDiagram
    autonumber
    participant CustomersBroker as Message Broker (customers topic)
    participant EventConsumer as orders CustomerAddressUpdatedConsumer
    participant IdempotencyStore as Idempotency Store (processedEvents)
    participant AddressRepo as CustomerAddressRepository

    CustomersBroker->>EventConsumer: CustomerAddressUpdated { addressId, customerId, street, city,   postalCode, reference, deleted }

    EventConsumer->>IdempotencyStore: isAlreadyProcessed(eventId)?
    IdempotencyStore-->>EventConsumer: false (new event)

    alt address exists in LRM
        EventConsumer->>AddressRepo: findByAddressId(addressId)
        AddressRepo-->>EventConsumer: CustomerAddress (existing record)
        EventConsumer->>AddressRepo: upsert(updatedAddress)
        Note over AddressRepo: UPDATE - apply new field values including deleted flag
    else new address
        EventConsumer->>AddressRepo: findByAddressId(addressId)
        AddressRepo-->>EventConsumer: null
        EventConsumer->>AddressRepo: upsert(newAddress)
        Note over AddressRepo: INSERT - new CustomerAddress record
    end

    AddressRepo-->>EventConsumer: ✓ upserted
    EventConsumer->>IdempotencyStore: markProcessed(eventId, TTL=14d)
    EventConsumer-->>CustomersBroker: ACK

    Note over EventConsumer: Redelivery scenario: IdempotencyStore returns true → skip processing → ACK
    Note over AddressRepo: deleted=true → address hidden at checkout (DELIVERY_ADDRESS_NOT_FOUND guard in UC-ORD-005)

sequenceDiagram
    autonumber
    actor Customer
    participant OrdersHandler as orders CheckoutHandler
    participant CartRepo as CartRepository
    participant AddrRepo as CustomerAddressRepository
    participant CatalogACL as CatalogACL (anti-corruption layer)
    participant CatalogAPI as catalog BC (internal API)
    participant OrderRepo as OrderRepository
    participant Outbox as Outbox (orders)

    Customer->>OrdersHandler: POST /carts/{cartId}/checkout { deliveryAddressId }

    OrdersHandler->>CartRepo: findById(cartId)
    CartRepo-->>OrdersHandler: Cart (OPEN, items=[p1×2, p2×1])

    Note over OrdersHandler: ownership check: cart.customerId == JWT.sub

    OrdersHandler->>AddrRepo: findByAddressId(deliveryAddressId)
    AddrRepo-->>OrdersHandler: CustomerAddress (deleted=false)

    Note over OrdersHandler: enforce ORD-RULE-002 (cart OPEN) enforce ORD-RULE-001 (not empty)

    OrdersHandler->>CatalogACL: validateProductsAndPrices([p1, p2])
    CatalogACL->>CatalogAPI: POST /internal/products/price-snapshot { productIds: [p1, p2] }
    CatalogAPI-->>CatalogACL: [{ productId:p1, name:.., price:15000 COP, status:ACTIVE },  { productId:p2, name:.., price:8500 COP, status:ACTIVE }]
    CatalogACL-->>OrdersHandler: List[PriceSnapshot]

    Note over OrdersHandler: All products ACTIVE ✓ Build OrderLineSnapshots Compute total = 38500 COP

    OrdersHandler->>OrdersHandler: cart.checkout() → status=CHECKED_OUT
    OrdersHandler->>OrdersHandler: Order.create(paymentMethod, addressId, addressSnapshot, total, lines) → emits OrderPlaced (in-memory)

    OrdersHandler->>CartRepo: save(cart CHECKED_OUT)
    OrdersHandler->>OrderRepo: save(order PENDING_PAYMENT)
    OrderRepo->>Outbox: INSERT OrderPlaced event

    OrdersHandler-->>Customer: 201 Created Location: /orders/{orderId}

    Note over Outbox: Outbox relay delivers OrderPlaced to message broker → inventory consumes
